# OpsPulse Mobile

This package is reserved for the React Native FieldAgent app.

Expo/React Native is intentionally not scaffolded yet. The mobile app will be
added when the project starts implementing FieldAgent login, assigned job lists,
offline queue storage, proof photo capture, location capture, QR scan, and sync.

Keeping this package as a placeholder today gives the monorepo its final product
shape without adding mobile generator noise to the foundation commit.
